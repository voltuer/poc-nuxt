# POC Nuxt

La idea aqui era probar un proyecto básico con Ant Design y Tailwind, y lograr que los compoenntes se pudieran usar sin
tener que importarlos individualmente (aprovechando las ventajas de Nuxt)

## Instalación

```bash
# pnpm
pnpm i
```

## Desarrollo

```bash
# pnpm
pnpm dev
```

## Prod

```bash
# pnpm
pnpm build
```
